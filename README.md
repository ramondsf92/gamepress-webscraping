# gamepress-webscraping
Web Scraping com Python para captura de dados sobre servos

Captura de dados sobre servos usando Requests e BeautifulSoup.
Na parte de Tier List, há um botão com a explicação do motivo para cada servo estar naquele tier.
Requests usado para capturar código-fonte da página e BeautifulSoup para captura dos textos através da localização dos div e filtragem de classe.
